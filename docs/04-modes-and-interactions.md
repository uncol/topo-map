# Режимы и взаимодействия

Документ описывает поведение пользовательских режимов и связанных ограничений.

## Режим `pan`

Назначение:
- перемещение viewport мышью.

Поведение:
- работает по blank и по элементам;
- отслеживание pointermove вынесено на `window` для непрерывного drag;
- `ViewportState` ограничивает `tx/ty`, чтобы карта не уходила за допустимые границы.

## Режим `zoomToArea`

Назначение:
- выделение прямоугольника и масштабирование в него.

Поведение:
- прямоугольник рисуется красным пунктиром;
- минимальный порог выделения учитывается (`10px` в screen-space);
- координаты выделения хранятся в local-space;
- рендер прямоугольника переводится в paper-space, чтобы клик и рамка совпадали при любом pan/zoom.

## Режим `edit`

Назначение:
- редактирование только перемещением существующих узлов.

Поведение:
- link move/creation отключены;
- перемещение узла считает смещение по real pointer (`clientX/clientY -> local`);
- при включенном snap позиция округляется к сетке;
- при выключенном snap перемещение свободное, без шагов сетки;
- позиция узла ограничена видимой областью (с учетом `boundsPadding`).

## Snap to grid

- включается `setSnapToGrid(true)`;
- выключается `setSnapToGrid(false)`;
- в edit режиме визуальная сетка показывается только когда snap включен.

## Guides

- включаются `setGuidesEnabled(true)`;
- выключаются `setGuidesEnabled(false)`;
- при выключении линии гарантированно очищаются;
- guides отображаются только во время drag в edit режиме;
- вычисления match выполняются в local-space с допуском в экранных пикселях, приведенным к текущему масштабу;
- рендер линий переводится в paper-space, чтобы совпадать с объектами при pan/zoom.

## Миникарта

Навигация:
- клик по minimap центрирует main viewport;
- drag viewport-рамки двигает main viewport.

Ограничения:
- рамка viewport клампится по границам minimap и не выходит за экран при экстремальных масштабах.

## Ограничения translate и padding

- `restrictTranslate` включен на paper.
- Дополнительно действует clamp через `ViewportState` + `DiagramService.getTranslateBounds()`.
- Параметр `boundsPadding` задает отступ от краев viewport:
- применяется для pan ограничений;
- применяется для ограничения перемещения узлов в edit.
